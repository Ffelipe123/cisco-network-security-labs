# 🔐 Cisco Network Security Labs

> Enterprise-grade portfolio focused on **Network Hardening**, **Segmentation**, and **Threat Mitigation** in Cisco-based infrastructures.

---

## 🎯 Objective

This repository demonstrates the implementation of **defensive security controls** in enterprise network environments using Cisco technologies.

The labs simulate **real-world business scenarios**, focusing on reducing the attack surface across:

* Layer 2 (Switching Security)
* Layer 3 (Segmentation & ACLs)
* Management Plane (Access Control & Monitoring)
* Edge Security (Firewall & VPN)

---

## 🧠 Approach

Each lab is designed following a **security-first mindset**, structured as:

1. **Business Context**
2. **Threat Modeling**
3. **Technical Implementation**
4. **Validation & Testing**
5. **Security Outcome**

This aligns with industry practices where network security is treated as a **risk management discipline**, not just configuration.

## 🏢 Simulated Business Environments

The labs represent realistic enterprise scenarios such as:

* Financial networks (high confidentiality)
* E-commerce infrastructures (availability & segmentation)
* Branch-office connectivity (secure WAN)
* Corporate LAN with insider threat risks

---

## 🔐 Security Controls Implemented

* Layer 2 Protection:

  * DHCP Snooping
  * Dynamic ARP Inspection (DAI)
  * Port Security

* Network Segmentation:

  * VLANs
  * Inter-VLAN Routing
  * Extended ACLs

* Management Plane Hardening:

  * SSH (secure access)
  * AAA authentication
  * Role-based access

* Monitoring & Visibility:

  * SNMPv3
  * Syslog
  * Time synchronization (NTP)

* Edge Security:

  * Zone-Based Firewall (ZBF)
  * IPsec VPN
