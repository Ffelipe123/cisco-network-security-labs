# 🔐 Cisco Network Security Labs (Packet Tracer Portfolio)

## 📌 Overview
This repository contains a structured portfolio of **hands-on network security labs** built using Cisco Packet Tracer.

The project focuses on combining **network engineering fundamentals** with **security best practices**, simulating real-world enterprise scenarios.

## 🎯 Objectives
- Design segmented and secure network architectures
- Implement Layer 2 and Layer 3 security controls
- Simulate common network attacks and mitigations
- Develop practical, job-ready networking skills

---

## 🧠 Skills Demonstrated

- VLAN Segmentation & Inter-VLAN Routing
- Access Control Lists (ACLs)
- Network Address Translation (NAT/PAT)
- Secure Remote Access (SSH)
- Layer 2 Security (Port Security, DHCP Snooping, ARP Inspection)
- Basic Network Hardening Techniques

---

## 🧪 Labs Included

| Lab | Title | Key Concepts |
|-----|------|-------------|
| 01 | Secure VLAN Segmentation | VLANs, Trunking, ACLs |
| 02 | Layer 2 Hardening | Port Security, DHCP Snooping, DAI |
| 03 | Secure Management | SSH, Device Hardening |
| 04 | Traffic Filtering | ACLs, NAT |
| 05 | Secure Enterprise Network | Full Integration |
